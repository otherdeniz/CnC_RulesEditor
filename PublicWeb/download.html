System Requirements

Minimum Requirements

  • OS: Windows 7 or later
  • Framework: .NET 6.0 Runtime (included in installer)
  • RAM: 2 GB
  • Storage: 100 MB free space
  • Display: 1024x768 resolution

Recommended

  • OS: Windows 10/11
  • RAM: 4 GB or more
  • Storage: 500 MB free space
  • Display: 1920x1080 resolution
  • Additional: Mouse with scroll wheel

Installation Instructions

Download the Editor

  • Click the "Download Latest Version" button above
  • Download the ZIP file from GitHub Releases
  • The file will be named something like CnC_RulesEditor_v2.6.60.zip

Extract the Files

  • Right-click the downloaded ZIP file
  • Select "Extract All..."
  • Choose a destination folder (e.g., C:\Tools\CnC_RulesEditor)
  • Click "Extract"

Run the Editor

  • Navigate to the extracted folder
  • Double-click Deniz.TiberiumSunEditor.Gui.exe
  • If prompted by Windows Defender, click "More info" then "Run anyway"
  • The editor will launch and you're ready to start modding!

Configure Your Game

  • Click the "Games" button in the editor
  • Select your C&C game (Tiberian Sun, Red Alert 2, etc.)
  • Browse to your game installation folder
  • Click "OK" to save the configuration

Auto-Update Feature

The C&C Rules Editor includes an integrated auto-update mechanism that checks for new versions on GitHub. When a new version is available, you'll be notified and can update with a single click.

Update Process

  • Editor automatically checks for updates on startup
  • Notification appears when new version is available
  • Click "Update" to download and install
  • Your settings and favorites are preserved
  • Manual update check available in Help menu
