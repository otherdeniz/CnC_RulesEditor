6. File Format Support

INI Files

  • Standard INI format with sections and key-value pairs
  • Comment preservation
  • Multi-line value support
  • Section inheritance (optional)
  • Automatic backup creation
  • Validation and error checking

MIX Files

  • Proprietary archive format
  • CRC-based file lookup
  • Optional Blowfish encryption
  • Supports nested MIX files
  • cache.mix, conquer.mix, local.mix (TS)
  • ra2.mix, ra2md.mix, language.mix (RA2)
  • Custom mod MIX files

SHP Files

  • Sprite sheet format
  • Frame-based animation
  • Palette-indexed colors
  • Compression support
  • Both TS and RA2 formats

Other Formats

  • AUD - Audio files
  • CSF - String tables
  • PAL - Color palettes
  • VXL - Voxel models
  • HVA - Voxel animations

Add Custom Mods

You can add any custom mod that is based on Tiberian Sun, Red Alert 2, or Dawn of the Tiberium Age. The editor will automatically detect the game type and load the appropriate configuration.

Custom Mod Features

  • Automatic game detection
  • Snapshot of original files
  • Mod-specific settings
  • Custom MIX file support
  • Preserve mod structure
  • Easy mod switching
