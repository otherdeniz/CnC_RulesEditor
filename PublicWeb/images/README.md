# Screenshots Directory

This directory should contain screenshots of the C&C Rules Editor application.

## Required Images

To complete the website, copy the following images to this directory:

1. **Buildings-TS.png** - Screenshot of the Tiberian Sun Buildings editor
2. **screenshot2.png** - Additional screenshot of the editor interface

## Source

Screenshots can be found in the main project's `Screenshots` folder:
- Copy from: `../Screenshots/Buildings-TS.png`
- Or download from: https://github.com/otherdeniz/CnC_RulesEditor/tree/master/Screenshots

## Image Guidelines

- Format: PNG
- Recommended size: 800x600 or larger
- Show key features of the editor
- Include cameos, property grids, and UI elements

## Current Status

The website is fully functional but will display broken image links until screenshots are added to this directory.
